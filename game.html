Game Rules
  • Luggages will move through the scanner one by one.
  • Some luggages contain illegal items, some do not.
  • When a luggage appears, decide if it should be Disposed or Passed:
    • Click Dispose if you think the luggage contains illegal items.
    • Click Pass if you think the luggage is clean.
  • You get 30 points for a correct answer, -10 points for a wrong answer or if you miss an illegal luggage.
  • If you score at least 30 points by the end, you win a coupon!
  • The game ends after a random number of rounds.
  • To restart the game you can reload the window.
